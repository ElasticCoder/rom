07808: THE 'POKE' COMMAND ROUTINE
The address of this routine is found in the parameter table.
In a similar manner to OUT, the POKE operation is performed.
Input
Output
POKE 07808 CALL TWO_PARAM The operands are fetched.
07811 LD (BC),A The actual POKE operation.
