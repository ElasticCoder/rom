1634: THE 'CHANNEL 'K' FLAG' SUBROUTINE
The address of this routine is derived from an offset found in the channel code lookup table.
Input
Output
CHAN_K 1634 SET 0,(IY+$02) Signal 'using lower screen'.
1638 RES 5,(IY+$01) Signal 'ready for a key'.
163C SET 4,(IY+$30) Signal 'using channel 'K''.
1640 JR CHAN_S_1 Jump forward.
