09A1: THE CASSETTE MESSAGES
Used by the routines at SAVE_ETC and SA_CONTRL.
Each message is given with the last character inverted (+80 hex.).
Input
Output
CASSETTE 09A1 DEFB $80 Initial byte is stepped over.
09A2 DEFM "Start tape, then press any key",$AE 'Start tape, then press any key.'
BLOCK_HDR 09C1 DEFM $0D,"Program:",$A0 Carriage return + 'Program: '
09CB DEFM $0D,"Number array:",$A0 Carriage return + 'Number array: '
09DA DEFM $0D,"Character array:",$A0 Carriage return + 'Character array: '
09EC DEFM $0D,"Bytes:",$A0 Carriage return + 'Bytes: '
