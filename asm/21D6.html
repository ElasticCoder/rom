21D6: THE 'IN-CHAN-K' SUBROUTINE
Used by the routine at INPUT.
This subroutine returns with the zero flag reset only if channel 'K' is being used.
Input
Output
IN_CHAN_K 21D6 LD HL,($5C51) The base address of the channel information for the current channel is fetched and the channel code compared to the character 'K'.
21D9 INC HL
21DA INC HL
21DB INC HL
21DC INC HL
21DD LD A,(HL)
21DE CP "K"
21E0 RET Return afterwards.
